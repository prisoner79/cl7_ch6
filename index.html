Question : Who created Python?

Answer : Python was created by Guido van Rossum in the late 1980s and was officially released in 1991. He designed Python to emphasize code readability and simplicity, making it one of the most popular programming languages today.

Question : How can we leave programmer's note in a Python program?

Answer : You can leave notes for yourself or other programmers using comments in Python. Comments are ignored during program execution. # for single-line and triple quotes for multi-line.

Example:
# Single-line comment
''' Multi-line comment '''

Question : Give the main features of Python which make it user-friendly.

Answer : Python is considered user-friendly due to:
1: Simple and readable syntax
2: Open-source and free to use
3: Platform independent
4: Large standard library
5: Interpreted language
6: Dynamically typed (No need to declare variable types)

Question : What is a comment? Mention its type with example.

Answer : Comments are used to explain code. They are not executed.

Types:
1. Single-line: # comment
2. Multi-line: ''' comment '''

Question : What are conditional statements?

Answer : Conditional statements control the flow of the program based on conditions.
They help in decision-making by checking conditions to perform certain actions.

Example :
age = 18
if age >= 18:
print("You are an adult")

Question : What are three conditional statements and mention example?

Answer : 1: if statement ,
Example:
if x > 0: print("Positive")

2: if-else statement,
Example:
x = 3
if x > 5:
print("x is greater than 5")
else:
print("x is not greater than 5")

3:if-elif-else statement
Example:
x = 5
if x > 5:
print("Greater")
elif x == 5:
print("Equal")
else:
print("Smaller")

Question : What is operator? How many types of operators are there in Python programming? Mention with example.

Answer : An operator is a symbol used to perform operations on variables and values.

Types: Arithmetic, Relational, Logical, Assignment, Membership
Example: 5 + 2, 5 > 2, 5 and 2

Question : What are variables? Mention its rules for assigning variable names.

Answer : Variables store data.

Rules:
1 : Must start with a letter (a–z, A–Z) or an underscore (_)
2 : Cannot start with a digit
3 : Can contain letters, digits, and underscores
4 : Cannot be a Python keyword
5 : Variable names are case-sensitive

Question : What is print function, input function, and type function? Mention example of each.

Answer : 1 : print() - outputs text. Example: print("Hello")
2 : input() - takes user input. Example: name = input("Enter your name: ")
3 : type() - returns type. Example: x = 10
print(type(x))

Question : What are reserved keywords in Python? Mention at least five keywords.

Answer : Reserved keywords have special meaning. Cannot use as variable names.

Examples: if, else, while, def, return
